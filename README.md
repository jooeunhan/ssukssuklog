# 🌱 쑥쑥로그 (SsukSsuk Log)

어린이집 교사용 아동 건강 관리 웹 서비스입니다.  
교사가 아동의 건강 상태를 기록하고, 보호자와 소통하며, 복약 요청 등을 효율적으로 관리할 수 있도록 기획한 프로젝트입니다.

---

## 📌 프로젝트 소개

쑥쑥로그는 어린이집 현장에서 발생하는 건강 기록 업무를 디지털화하여  
교사의 업무 부담을 줄이고, 보호자와의 소통을 더 원활하게 만드는 것을 목표로 합니다.

---

## ✨ 주요 기능

- 로그인 / 회원가입 UI
- 교사용 대시보드
- 아동 목록 및 반별 조회
- 건강 상태 입력 (체온, 증상, 메모)
- 아동 건강 히스토리 확인
- 보호자 알림 메시지 관리
- 복약 요청 및 투약 여부 관리

---

## 🛠 사용 기술

- React
- Vite
- Styled-components
- JavaScript (ES6+)
